# Rewrite Lens

Use esta lente para transformar bagagem técnica em linguagem forte de currículo.

## Bullet Formula

`problema + ação + stack + escopo + resultado + ownership`

Nem tudo precisa caber em um bullet, mas a experiência como um todo precisa cobrir essas dimensões.

## Technical Specialist Test

Pergunte:

- esse texto usa termos corretos do domínio?
- ele distingue bem framework de problema real?
- mostra produção, qualidade e tradeoff quando isso importa?
- alguém da área reconheceria profundidade aqui?

## HR Specialist Test

Pergunte:

- o valor aparece rápido?
- a senioridade está clara?
- a stack principal está visível?
- o texto é legível e triável?

## Common Failure Modes

- tarefa sem impacto
- stack sem contexto
- escopo sem contribuição individual
- senioridade declarada, não provada
- linguagem inflada ou genérica
- buzzwords sem aderência real

## Strong Verbs By Intent

- build
- design
- implement
- migrate
- optimize
- scale
- stabilize
- automate
- secure
- mentor
- lead

## Translation Rule

Troque:

- “trabalhei com”
- “atuei em”
- “participei de”

Por linguagem que mostre:

- decisão
- execução
- resultado
- responsabilidade
