# Output Structure

The final file must be a single markdown document named `GUIA_FINAL_MELHORIA_LINKEDIN.md`.

## Required Sections

1. Executive summary
2. What the profile communicates today
3. What to change in each microcomponent
4. Skills by microcomponent
5. Skills to learn or deepen
6. Gaps versus the current market
7. Practical rewrite
8. Proof to build
9. Prioritized action plan
10. Final review checklist

## Required Tables

Use tables where useful for:

- microcomponent
- current score
- target score
- gap
- impact
- priority
- effort
- recommendation

Also include a consolidated component score table near the top of the file.

And for skills:

- skill
- type
- urgency
- market value
- career fit
- where it should appear on LinkedIn
- how to prove it

## Microcomponents To Cover

- photo
- banner
- name and professional identification
- headline
- custom URL
- location and geography
- industry
- About
- Featured
- Activity / content
- Experience
- Skills
- Recommendations
- Projects
- Certifications
- Education
- Other relevant sections

## Rewrite Requirements

For each relevant section, include:

- current diagnosis
- weakness
- impact
- what to change
- what it should look like
- skill needed
- priority

Also include:

- what to remove or reduce
- what proof is missing
- what should be added only after evidence exists

## Priority Labels

Use exactly these labels:

- critical
- high
- medium
- low
